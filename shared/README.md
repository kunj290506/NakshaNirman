# Shared Assets

This directory contains shared resources used across the application.

## Structure

```
shared/
├── materials/       # PBR material textures
├── furniture/       # 3D furniture models (GLTF)
├── hdri/           # HDRI environment maps
└── templates/      # CAD templates
```

## Materials

PBR textures for:
- Walls (plaster, brick, wood)
- Floors (concrete, hardwood, tile)
- Windows (glass, metal frames)

## Furniture

Pre-made 3D models for:
- Sofas, chairs, tables
- Beds, wardrobes
- Kitchen appliances
- Bathroom fixtures

## HDRI

Environment maps for realistic lighting:
- Sunny day
- Cloudy
- Evening
- Night

## Templates

CAD template files:
- Title blocks
- Layer configurations
- Dimension styles
